# Dict_exercise
这是一个用C语言制作的字典小程序(练习项目)

## 一、功能介绍
### 1.界面
- 1.1有简单的菜单显示
- 1.2给予每步操作文字反馈

### 2.特殊命令：
- 2.1@menu:显示菜单
- 2.2@exit:退出程序

### 3.功能
- 3.1英译中
- 3.2中译英
